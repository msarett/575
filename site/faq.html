COMP 575: Fall 2015

Frequently Asked Questions

Where is the code for each assignment?

/afs/cs.unc.edu/project/courses/comp575-f15/code/
Each assignment is in its own pa* directory

e.g. The first assignment is found in /afs/cs.unc.edu/project/courses/comp575-f15/code/pa0

Where do I store my code for each assignment?

/afs/cs.unc.edu/project/courses/comp575-f15/users/your-name
Each assigment should have its own pa* directory

e.g. Your first assignment should be stored in /afs/cs.unc.edu/project/courses/comp575-f15/users/your-name/pa0

Can I use the department's computers to write/compile my code?

Yes. 
    ssh -X your-name@classroom.cs.unc.edu
    cd /afs/cs.unc.edu/project/courses/comp575-f15/users/your-name
If you need to start on a new assignment (e.g. pa# where # is 0 or 1 or whatever), you'll need to first copy the source from code into your local dir 
    cp -R ../code/pa# .
Once you have copied it, you can just work on it in your local copy in your pa# directory

If you need an account, contact John Sopko sopko@cs.unc.edu

Do I have to use the department's computers?

No. The only actual requirements are:
  • You store your final code for each assignment in /afs/cs.unc.edu/project/courses/comp575-f15/users/your-name/pa#
  • It can compile successfully using the department's computers (i.e. I can compile and run it).
Developing locally (on your laptop) is perfectly fine, and may be faster than doing all your work via ssh.

How do I compile and run the assignment?

Assuming you're in your directory on /afs... 
    > cd pa0
    > make
    > ./image
You show now have created the PNG files (try 'ls *.png' to see).
If your remote connection was created with -X (i.e. ssh -X ...) and your local terminal supports X Windows, you can try viewing the files with 'preview' 
    > ../../../code/preview *.png
If that doesn't work, you can always copy the image files to your local machine (laptop) 
    # from your local machine
    > scp your-name@classroom.cs.unc.edu:/afs/cs.unc.edu/project/courses/comp575-f15/users/your-name/pa0/*.png .

What if my laptop is Windows?

Go to software.unc.edu and download the "SSH/SFTP Secure Shell"

I'm a grad student. Will I get "credit" for 575?

Grad students can get credit for any course numbered greater than 400, so yes.
